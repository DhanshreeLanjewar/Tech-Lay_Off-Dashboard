# Tech Layoffs Dashboard (2020–2025)

This Power BI dashboard analyzes tech layoffs across different companies, industries, and countries from 2020 to 2025 using real-world layoff data. The dashboard provides interactive insights into layoff trends, company impact, industry distribution, and geographic analysis through dynamic visualizations and KPI cards.

## Tools Used
Power BI
DAX
Data Visualization

## Key Insights
Year-wise layoff trends
Company & industry analysis
Country-wise layoffs
Interactive filters and KPIs
